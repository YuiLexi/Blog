11.1 对弧长的曲线积分

11.1.1 对弧长的曲线积分的概念与性质

定义（曲线积分）： 曲线积分表达式：

∫Lf(x,y)dL\displaystyle \int\limits_{L}f(x,y)\mathrm{d}L L∫​f(x,y)dL

性质：

  1. ∫L[αf(x,y)+βg(x,y)]dL=α∫Lf(x,y)dL+β∫Lg(x,y)dL\displaystyle \int\limits_{L}\left [ \alpha f(x,y)+\beta g(x,y) \right ] \mathrm{d}L= \alpha\int\limits_{L}f(x,y)\mathrm{d}L+\beta \int\limits_{L}g(x,y)\mathrm{d}LL∫​[αf(x,y)+βg(x,y)]dL=αL∫​f(x,y)dL+βL∫​g(x,y)dL 。
  2. $\displaystyle \int\limits_{L}f(x,y) \mathrm{d}L = \int\limits_{L_{1} }f(x,y) \mathrm{d}L +\int\limits_{L_{2} }f(x,y) \mathrm{d}L $ 。
  3. ∣∫Lf(x,y)dL∣≤∫L∣f(x,y)∣dL\displaystyle \left | \int\limits_{L}f(x,y)\mathrm{d}L \right | \le \int\limits_{L}\left | f(x,y) \right | \mathrm{d}L∣∣∣∣∣∣∣​L∫​f(x,y)dL∣∣∣∣∣∣∣​≤L∫​∣f(x,y)∣dL 。
  4. if:f(x,y)≤g(x,y),∫Lf(x,y)dL≤∫Lg(x,y)dL\displaystyle \text{if:} f(x,y)\le g(x,y) ,\quad \int\limits_{L}f(x,y) \mathrm{d}L\le \int\limits_{L}g(x,y) \mathrm{d}Lif:f(x,y)≤g(x,y),L∫​f(x,y)dL≤L∫​g(x,y)dL 。

11.1.2 对弧长的曲线积分的计算方法

定理：假设曲线 LLL 的参数方程为 $\left{\begin{matrix} x=\varphi (t)\ y=\psi (t)\end{matrix}\right.\quad (\alpha \le t\le \beta ) $ ，并且 [φ′(t)]2+[ψ′(t)]2≠0\displaystyle \left [ {\varphi }' (t) \right ]^{2}+ \left [ {\psi}' (t) \right ]^{2}\ne 0[φ′(t)]2+[ψ′(t)]2​=0 ，那么

∫Lf(x,y)dL=∫αβf[φ(t),ψ(t)][φ′(t)]2+[ψ′(t)]2dt\displaystyle \int\limits_{L}f(x,y)\mathrm{d}L= \int\limits_{\alpha }^{\beta } f\left [ \varphi (t),\psi (t)\right ] \sqrt{\displaystyle\left [ {\varphi }'(t)\right ]^{2}+\left [ {\psi}' (t)\right ]^{2}}\mathrm{d}t L∫​f(x,y)dL=α∫β​f[φ(t),ψ(t)][φ′(t)]2+[ψ′(t)]2​dt

特殊情况：

  1. 如果 y=g(x)y=g(x)y=g(x) ，那么参数方程为：$\displaystyle \left{\begin{matrix} x = t\y=g(t)\end{matrix}\right.\quad t\in \left [ m,n \right ] $ 。

11.2 对坐标的曲线积分

11.2.1 对坐标的曲线积分的概念与性质

引例：物体做功的计算公式为： $W = \overrightarrow{F} \cdot \overrightarrow{x} $ ，假设物体的运动轨迹是一条曲线 F(x,y)=0F(x,y)=0F(x,y)=0 进行运动，那么将曲线分成 nnn 段，每一段近似为直线运动，去每一段上的一点作为计算点，得到：

\begin{align}W & = \displaystyle \lim_{\lambda \to 0} \sum_{i=1}^{n} F(\xi _{i},\eta _{i})\cdot (\Delta x\overrightarrow{i} +\Delta y\overrightarrow{j})\\& = \lim_{\lambda \to 0} \sum_{i = 1}^{n}\left [ P(\xi _{i},\eta _{i})\Delta x_{i} +Q(\xi _{i},\eta _{i})\Delta y_{i} \right ] \end{align}

其中， $F(x,y)= P(x,y)\overrightarrow{i} + Q(x,y)\overrightarrow{j} $ 。

定义：设 LLL 为有向线段，函数 F(x,y)F(x,y)F(x,y) 的对坐标轴的曲线积分为：

\begin{align}\displaystyle \int\limits_{L}P(x,y)\mathrm{d}x &=\lim_{\lambda \to 0} \sum_{i = 1}^{n}P(\xi _{i},\eta _{i})\Delta x_{i} \\\displaystyle \int\limits_{L}Q(x,y)\mathrm{d}y &=\lim_{\lambda \to 0} \sum_{i = 1}^{n}Q(\xi _{i},\eta _{i})\Delta x_{i} \\\end{align}

三维的曲线同样适用。

性质 1： ∫L[αF1(x,y)+βF2(x,y)]⋅dr=α∫LF1(x,y)⋅dr+β∫LF2(x,y)⋅dr\displaystyle \int\limits_{L}\left [ \alpha F_{1} (x,y)+ \beta F_{2}(x,y) \right ] \cdot \mathrm{d}r =\alpha \int\limits_{L} F_{1} (x,y) \cdot \mathrm{d}r+\beta \int\limits_{L} F_{2} (x,y) \cdot \mathrm{d}rL∫​[αF1​(x,y)+βF2​(x,y)]⋅dr=αL∫​F1​(x,y)⋅dr+βL∫​F2​(x,y)⋅dr 。

性质 2： $\int\limits_{L}F(x,y)\cdot \mathrm{d}r = \int\limits_{L_{1}}F(x,y)\cdot \mathrm{d}r +\int\limits_{L_{2}}F(x,y)\cdot \mathrm{d}r $ 。

性质 3： L−L^{-}L− 是曲线 LLL 的反向曲线，那么 $\int\limits_{L^{-} }F(x,y)\cdot \mathrm{d}r = -\int\limits_{L}F(x,y)\cdot \mathrm{d}r $ 。

11.2.2 对坐标的曲线积分的计算方法

定理：函数 F(x,y)F(x,y)F(x,y) 能表示为参数方程： {x=φ(t)y=ψ(t)\left\{\begin{matrix}x=\varphi (t)\\y=\psi (t)\end{matrix}\right.{x=φ(t)y=ψ(t)​ ，起点 AAA 到终点 BBB ，参数满足 $t:\alpha\to \beta $ ，那么得到下面的关系：

\begin{align}\int\limits_{L}P(x,y)\mathrm{d}x+Q(x,y)\mathrm{d}y & = \int_{\alpha }^{\beta } \left [ P \left ( \varphi(t),\psi (t) \right ) {\varphi}'(t)+ Q\left ( \varphi(t),\psi (t) \right ) {\psi}'(t) \right ] \mathrm{d}t \end{align}

11.2.3 两类曲线积分之间的联系

回忆：

  • 第一类曲线积分： ∫Lf(x,y)dL=∫αβf[φ(t),ψ(t)][φ′(t)]2+[ψ′(t)]2dt\displaystyle \int\limits_{L}f(x,y)\mathrm{d}L= \int\limits_{\alpha }^{\beta } f\left [ \varphi (t),\psi (t)\right ] \sqrt{\displaystyle\left [ {\varphi }'(t)\right ]^{2}+\left [ {\psi}' (t)\right ]^{2}}\mathrm{d}tL∫​f(x,y)dL=α∫β​f[φ(t),ψ(t)][φ′(t)]2+[ψ′(t)]2​dt 。
  • 第二类曲线积分： \begin{align}\int\limits_{L}P(x,y)\mathrm{d}x+Q(x,y)\mathrm{d}y & = \int_{\alpha }^{\beta } \left [ P \left ( \varphi(t),\psi (t) \right ) {\varphi}'(t)+ Q\left ( \varphi(t),\psi (t) \right ) {\psi}'(t) \right ] \mathrm{d}t \end{align} 。

上面两种曲线积分，其本质上应该是等价的，那么可以得到：

\begin{align}\text{第二类} &=\int_{\alpha }^{\beta } \left [ P \left ( \varphi(t),\psi (t) \right ) {\varphi}'(t)+Q\left ( \varphi(t),\psi (t) \right ) {\psi}'(t) \right ]\mathrm{d}t \\&=\int_{\alpha }^{\beta}P(\varphi(t),\psi (t))\frac{ {\varphi}'(t)}{\sqrt{ {\varphi}'^{2} (t)+{\psi}'^{2} (t)} } +Q(\varphi(t),\psi (t))\frac{ {\psi}'(t)}{\sqrt{ {\varphi}'^{2} (t)+{\psi}'^{2} (t)}} \sqrt{ {\varphi}'^{2} (t)+{\psi}'^{2} (t)}\mathrm{d}t\\&=\int\limits_{L}\left [ P(x,y)\cos \alpha +Q(x,y)\cos \beta \right ] \mathrm{d}s \end{align}

11.3 格林公式及其应用

11.3.1 格林公式

在平面闭区域 DDD 上的二重积分，只关注平面的边界曲线，就是格林公式的意义。

单连通区域： DDD 平面区域，其中内部任意的闭曲线围成的部分都是属于 DDD 。

复连通区域：区域中间有洞的。

正方向：外边缘逆时针，内边缘顺时针。

定理（格林公式）：闭区域 DDD 有分段光滑的曲线 LLL 围成的，函数 P(a,y),Q(x,y)P(a,y),Q(x,y)P(a,y),Q(x,y) 在 DDD 上有一阶连续偏导，那么有：

∬D(∂Q∂x−∂P∂y)dxdy=∮LPdx+Qdy\iint\limits_{D}(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y})\mathrm{d}x\mathrm{d}y = \oint_{L}P\mathrm{d}x+Q\mathrm{d}y D∬​(∂x∂Q​−∂y∂P​)dxdy=∮L​Pdx+Qdy

11.4 对面积的曲面积分

11.4.1 对面积的曲面积分的概念与性质

引例：设曲面形构建具有连续面密度 ρ(x,y,z)\rho (x,y,z)ρ(x,y,z) ，求质量 MMM 。

定义：设 $\Sigma $ 为光滑曲面， f(x,y,z)f(x,y,z)f(x,y,z) 是定义在 $\Sigma $ 上的一个有界函数，若对 $\Sigma $ 做任意分割和局部区域取任意点，下面

lim⁡λ→0∑k=1nf(ξk,ηk,ζk)ΔSk=∬Σf(x,y,z)dS\lim_{\lambda \to 0} \sum_{k=1}^{n}f(\xi_{k} ,\eta_{k} ,\zeta_{k})\Delta S_{k} =\iint\limits_{\Sigma }f(x,y,z)\mathrm{d}S λ→0lim​k=1∑n​f(ξk​,ηk​,ζk​)ΔSk​=Σ∬​f(x,y,z)dS

都存在，则称此极限为函数 f(x,y,z)f(x,y,z)f(x,y,z) 在曲面 $\Sigma $ 上对面积的曲面积分或者第一类曲面积分，其中 f(x,y,z)f(x,y,z)f(x,y,z) 叫做被积函数， $\Sigma $ 叫做积分曲面。

曲面积分的性质：

  1. 与曲线积分的性质相似。

11.4.2 对面积的曲面积分的计算法

定理：设有光滑曲面 $\Sigma :z=z(x,y),\quad (x,y)\in D_{xy} $ ，函数 f(x,y,z)f(x,y,z)f(x,y,z) 在 $\Sigma $ 上连续，则曲面积分 ∬Σf(x,y,z)dS\displaystyle \iint\limits_{\Sigma }f(x,y,z)\mathrm{d}SΣ∬​f(x,y,z)dS 存在，且有：

∬Σf(x,y,z)dS=∬Df(x,y,z(x,y))1+zx2(x,y)+zy2(x,y)dxdy\iint\limits_{\Sigma }f(x,y,z)\mathrm{d}S = \iint\limits_{D}f(x,y,z(x,y))\sqrt{1+z_{x}^{2}(x,y)+z_{y}^{2}(x,y)} \mathrm{d}x\mathrm{d}y Σ∬​f(x,y,z)dS=D∬​f(x,y,z(x,y))1+zx2​(x,y)+zy2​(x,y)​dxdy

上述公式是：曲面向 xOyxOyxOy 平面进行投影，当然也可以向 xOz,yOzxOz,yOzxOz,yOz 平面投影。

11.5 对坐标的曲面积分

11.5.1 对坐标的曲面积分的概念与性质

有向曲面：

  1. 方向余弦 cos⁡α>0\cos \alpha >0cosα>0 ，作为曲面的前侧； cos⁡α<0\cos \alpha<0cosα<0 ，作为曲面的后侧。
  2. 方向余弦 cos⁡β>0\cos\beta>0cosβ>0 ，作为曲面的右侧； cos⁡β<0\cos\beta <0cosβ<0 ，作为曲面的左侧。
  3. 方向余弦 cos⁡γ>0\cos\gamma >0cosγ>0 ，作为曲面的上侧； cos⁡γ<0\cos\gamma <0cosγ<0 ，作为曲面的下册。
  4. 封闭曲面：内测、外侧。

设 $\Sigma $ 为空间中的有向曲面，其面元 ΔS\Delta SΔS 在 xOyxOyxOy 面上的投影记为 $\left ( \Delta S \right ) _{xy} $ 的面积为 $\left ( \Delta \sigma \right ) _{xy} $ ，则规定：

(ΔS)xy={(Δσ)xycos⁡γ>0−(Δσ)xycos⁡γ<00cos⁡γ=0\left ( \Delta S\right)_{xy} =\left\{\begin{matrix} \left ( \Delta \sigma \right)_{xy} & \cos \gamma >0\\ -\left ( \Delta \sigma \right)_{xy} & \cos \gamma <0\\ 0&\cos \gamma =0\end{matrix}\right. (ΔS)xy​=⎩⎪⎨⎪⎧​(Δσ)xy​−(Δσ)xy​0​cosγ>0cosγ<0cosγ=0​

定义：设 $\Sigma $ 为光滑的有向曲面，在 $\Sigma $ 上定义了一个向量场 A→=(P,Q,R)\overrightarrow{A} =(P,Q,R)A=(P,Q,R) ，其中， P,Q,RP,Q,RP,Q,R 是关于 x,y,zx,y,zx,y,z 的三元函数，那么

∬ΣPdydz+Qdxdz+Rdxdy\iint\limits_{\Sigma }P\mathrm{d}y\mathrm{d}z+Q\mathrm{d}x\mathrm{d}z+R\mathrm{d}x\mathrm{d}y Σ∬​Pdydz+Qdxdz+Rdxdy

称为对坐标的曲面积分或者第二类曲面积分。

∬ΣPdydz\displaystyle \iint\limits_{\Sigma }P\mathrm{d}y\mathrm{d}zΣ∬​Pdydz 称为 PPP 在有向曲面 $\Sigma $ 上 对 x,yx,yx,y 的曲面积分；

∬ΣQdxdz\displaystyle \iint\limits_{\Sigma }Q\mathrm{d}x\mathrm{d}zΣ∬​Qdxdz 称为 QQQ 在有向曲面 $\Sigma $ 上 对 x,zx,zx,z 的曲面积分；

∬ΣRdxdy\displaystyle \iint\limits_{\Sigma }R\mathrm{d}x\mathrm{d}yΣ∬​Rdxdy 称为 RRR 在有向曲面 $\Sigma $ 上 对 y,zy,zy,z 的曲面积分。

流过有向曲面 $\Sigma $ 的流体的流量为

Φ=∬ΣPdydz+Qdxdz+Rdxdy\Phi =\iint\limits_{\Sigma }P\mathrm{d}y\mathrm{d}z+Q\mathrm{d}x\mathrm{d}z+R\mathrm{d}x\mathrm{d}y Φ=Σ∬​Pdydz+Qdxdz+Rdxdy

如果记 $\Sigma $ 正常的单位法向量为 n→=(cos⁡α,cos⁡β,cos⁡γ)\overrightarrow{n} =(\cos \alpha ,\cos \beta ,\cos \gamma )n=(cosα,cosβ,cosγ) ，并且令 dS→=n→dS=(dydz,dxdz,dxdy)\overrightarrow{\mathrm{d}S} =\overrightarrow{n}\mathrm{d}S=(\mathrm{d}y\mathrm{d}z,\mathrm{d}x\mathrm{d}z,\mathrm{d}x\mathrm{d}y)dS=ndS=(dydz,dxdz,dxdy) ，则对坐标的曲面积分也常写为如下向量形式：

∬ΣPdydz+Qdxdz+Rdxdy=∬ΣA→⋅dS→\iint\limits_{\Sigma }P\mathrm{d}y\mathrm{d}z+Q\mathrm{d}x\mathrm{d}z+R\mathrm{d}x\mathrm{d}y=\iint\limits_{\Sigma }\overrightarrow{A} \cdot \overrightarrow{\mathrm{d}S} Σ∬​Pdydz+Qdxdz+Rdxdy=Σ∬​A⋅dS

11.5.2 对坐标的曲面积分的计算法

定理：设光滑曲面 $\Sigma :z=z(x,y),\quad (x,y)\in D_{xy} $ ，取上侧 R(x,y,z)R(x,y,z)R(x,y,z) 是 $\Sigma $ 上的连续函数，则

∬ΣR(x,y,z)dxdy=∬DxyR(x,y,z(x,y))dxdy\iint\limits_{\Sigma }R(x,y,z)\mathrm{d}x\mathrm{d}y=\iint\limits_{D_{xy} }R(x,y,z(x,y))\mathrm{d}x\mathrm{d}y Σ∬​R(x,y,z)dxdy=Dxy​∬​R(x,y,z(x,y))dxdy

11.5.3 两类曲面积分之间的联系

∬ΣPdydz+Qdzdx+Rdxdy=∬Σ(Pcos⁡α+Qcos⁡β+Rcos⁡γ)dS\iint\limits_{\Sigma }P\mathrm{d}y\mathrm{d}z+Q\mathrm{d}z\mathrm{d}x+R\mathrm{d}x\mathrm{d}y=\iint\limits_{\Sigma }(P\cos \alpha +Q\cos \beta +R\cos \gamma )\mathrm{d}S Σ∬​Pdydz+Qdzdx+Rdxdy=Σ∬​(Pcosα+Qcosβ+Rcosγ)dS
