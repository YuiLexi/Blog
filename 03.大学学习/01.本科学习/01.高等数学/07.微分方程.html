7.1 微分方程的基本概念

一般地，凡是表示未知函数、未知函数的导数与自变量之间的关系的方程，叫做**微分方程**，有时简称方程。

微分方程中所出现的未知函数的最高阶导数的阶数，叫做**微分方程的阶**。

把某个函数带入微分方程，能使该方程成为恒等式，这个函数就叫做该**微分方程的解**。

如果为微分方程的解中含有任意常数，且任意常数的个数与微分方程的阶数相同，这样的解叫做微分方程的**通解**。

初值条件。

确定了通解中任意常数以后，就得到微分方程的**特解**。

7.2 可分离变量的微分方程

一阶微分方程：

y′=f(x,y)\displaystyle {y}' =f(x,y) y′=f(x,y)

一阶微分方程有时也写成如下的对称形式：

P(x,y)dx+Q(x,y)dy=0\displaystyle P(x,y)\mathrm{d}x +Q(x,y)\mathrm{d}y =0 P(x,y)dx+Q(x,y)dy=0

🧭一般地，如果一个一阶微分方程能写成🧭🧭🧭

g(y)dy=f(x)dx\displaystyle g(y)\mathrm{d}y =f(x)\mathrm{d}x g(y)dy=f(x)dx

的形式，就是说，能把微分方程写成一端只含 yyy 的函数的 dy\mathrm{d}ydy ，另一端只含 xxx 的函数和 $\mathrm{d}x $ 那么原方程就成为可分离变量的微分方程。

假定方程中的 g(x)g(x)g(x) 与 f(x)f(x)f(x) 函数是连续的。设 y=φ(x)y=\varphi (x)y=φ(x) 是方程的解，将它带入方程得

g[φ(x)]φ′(x)dx=f(x)dx\displaystyle g\left [ \varphi (x) \right ] {\varphi }' (x)\mathrm{d}x =f(x)\mathrm{d}x g[φ(x)]φ′(x)dx=f(x)dx

7.3 齐次方程

7.3.1 齐次方程

如果一阶微分方程可化为

dydx=φ(yx)\displaystyle \frac{\mathrm{d}y}{\mathrm{d}x} =\varphi (\frac{y}{x} ) dxdy​=φ(xy​)

的形式，那么就称这方程为**齐次方程**，例如

(xy−x2)dx−(x2−2xy)dy=0\displaystyle (xy-x^{2} )\mathrm{d}x -(x^{2}-2xy )\mathrm{d}y =0 (xy−x2)dx−(x2−2xy)dy=0

齐次方程的解题：

  1. $\displaystyle u=\frac{y}{x} $ ；
  2. y=xu\displaystyle y=xuy=xu ；
  3. $\displaystyle \frac{\mathrm{d} y}{\mathrm{d} x} =u + x\frac{\mathrm{d} u}{\mathrm{d} x} $ ；
  4. 带入方程。

7.3.2 可化为齐次的方程

dydx=ax+by+ca1x+b1y+c1\displaystyle \frac{\mathrm{d} y}{\mathrm{d} x} =\frac{ax+by+c}{a_{1} x+b_{1} y+c_{1} } dxdy​=a1​x+b1​y+c1​ax+by+c​

当 c=c1=0c = c_{1} =0c=c1​=0 时，方程是齐次的；否则不是齐次的。在非齐次的情况下，可用下列变换把它化为齐次方程，令 x=X+h,y=Y+kx =X+h,y=Y+kx=X+h,y=Y+k ，其中 hhh 以及 kkk 是待定常系数。于是

dYdX=aX+bY+ah+bk+ca1X+b1Y+a1h+b1k+c1\displaystyle \frac{\mathrm{d} Y}{\mathrm{d} X} =\frac{aX+bY+ah+bk+c}{a_{1} X+b_{1} Y+a_{1} h+b_{1} k+c_{1} } dXdY​=a1​X+b1​Y+a1​h+b1​k+c1​aX+bY+ah+bk+c​

如果方程组

的系数行列式 ∣aba1b1∣≠0\displaystyle \begin{vmatrix} a &b \\ a_{1} &b_{1} \end{vmatrix}\ne 0∣∣∣∣∣​aa1​​bb1​​∣∣∣∣∣​=0 ，方程有唯一解。那么可以给定 hhh 和 kkk 使它们满足上述方程组。这样，微分方程就化为

dYdX=aX+bYa1X+b1Y\displaystyle \frac{\mathrm{d} Y}{\mathrm{d} X} =\frac{aX+bY}{a_{1} X+b_{1} Y} dXdY​=a1​X+b1​YaX+bY​

7.4 一阶线性微分方程

7.4.1 线性方程

方程

dydx+P(x)y=Q(x)\displaystyle \frac{\mathrm{d}y }{\mathrm{d}x } +P(x)y=Q(x) dxdy​+P(x)y=Q(x)

叫做**一阶线性微分方程，因为他对于未知函数 yyy 及其导数是一次方程。如果 Q≡0\displaystyle Q\equiv 0Q≡0 ，那么方程变为齐次的**；反之，则为非齐次的。

一阶线性方程的求解：

设 dydx+P(x)y=Q(x)\displaystyle \frac{\mathrm{d}y }{\mathrm{d}x } +P(x)y=Q(x)dxdy​+P(x)y=Q(x) 为非齐次线性方程。为了求出该方程的解，我们先利用齐次形式

dydx+P(x)y=0\displaystyle \frac{\mathrm{d}y }{\mathrm{d}x } +P(x)y=0 dxdy​+P(x)y=0

上述方程叫做对应于非齐次线性方程的齐次线性方程1.然后，分离变量可得

dyy=−P(x)dx\displaystyle \frac{\mathrm{d}y }{y }=-P(x)\mathrm{d}x ydy​=−P(x)dx

两边积分，然后求解，得

y=Ce−∫P(x)dx(C=±eC1)y = \displaystyle C{\large \mathrm{e} ^{-\int P(x)\mathrm{d}x }} (C = \pm \mathbin{e}^{C_{1} } ) y=Ce−∫P(x)dx(C=±eC1​)

这就是对应的齐次线性方程的通解。

现在我们使用所谓的常数易变法来求解非齐次线性方程的通解。该方法是将齐次方程的通解中的 CCC 换成未知函数 u(x)u(x)u(x) ，即做变换

y=ue−∫P(x)dxy = \displaystyle u{\large \mathrm{e} ^{-\int P(x)\mathrm{d}x }} y=ue−∫P(x)dx

带入非齐次线性方程得

u=∫Q(x)e∫P(x)dxdx+Cu = \displaystyle \int Q(x)\mathrm{e}^{\int P(x)\mathrm{d}x } \mathrm{d}x +C u=∫Q(x)e∫P(x)dxdx+C

因此，非齐次线性方程的通解为

\begin{align}y & = \mathrm{e}^{-\int P(x)\mathrm{d}x } ( \int Q(x)\mathrm{e}^{\int P(x)\mathrm{d}x } \mathrm{d}x +C)\\& = C\mathrm{e}^{-\int P(x)\mathrm{d}x } + \mathrm{e}^{-\int P(x)\mathrm{d}x }\int Q(x)\mathrm{e}^{\int P(x)\mathrm{d}x } \mathrm{d}x\end{align}

上式右端第一项对应的是齐次线性方程的通解，第二项是非齐次线性方程的一个特解。由此可知，一阶非齐次线性方程的通解等于对应的齐次方程的通解与非齐次方程的一个特解之和。

7.4.2 伯努利方程

方程

dydx+P(x)y=Q(x)yn(n≠0,1)\displaystyle \frac{\mathrm{d}y }{\mathrm{d}x } +P(x)y=Q(x)y^{n} \quad (n\ne 0,1) dxdy​+P(x)y=Q(x)yn(n=0,1)

叫做**伯努利方程**。当 n=0,1n=0,1n=0,1 时，这时线性微分方程；当 n≠0,1n\ne 0,1n=0,1 时，方程不是线性的，但是通过变量的代换，便可把它化为线性。方程两边同时除以 yny^{n}yn 得

y−ndydx+P(x)y1−n=Q(x)\displaystyle y^{-n} \frac{\mathrm{d}y }{\mathrm{d}x } +P(x)y^{1-n} =Q(x) y−ndxdy​+P(x)y1−n=Q(x)

令 $z = y^{1-n} $ ，所以 $\displaystyle \frac{\mathrm{d} z}{\mathrm{d} x} =(1-n)y^{-n} \frac{\mathrm{d} y}{\mathrm{d} x} $ ，伯努利方程就能线性化为

dzdx+(1−n)P(x)z=(1−n)Q(x)\displaystyle \frac{\mathrm{d} z}{\mathrm{d} x} +(1-n)P(x)z=(1-n)Q(x) dxdz​+(1−n)P(x)z=(1−n)Q(x)

求出这个方程得通解后，反代换就能得到伯努利方程的通解。

7.5 可降阶得高阶微分方程

二阶及二阶以上的微分方程，称高阶微分方程。对于有些高阶微分方程，我们可以通过代换将它化成较低阶的方程来求解。

7.5.1 y(n)=f(x){y}^{(n)} =f(x)y(n)=f(x) 型的微分方程

微分方程

y(n)=f(x){y}^{(n)} =f(x) y(n)=f(x)

的右端仅含有自变量 xxx 。显然，两边积分，就能得到一个 n−1n-1n−1 阶的微分方程

y(n−1)=∫f(x)dx+C1{y}^{(n-1)} =\int f(x)\mathrm{d}x +C_{1} y(n−1)=∫f(x)dx+C1​

同理可得

y(n−2)=∫[∫f(x)dx+C1]dx+C2{y}^{(n-2)} =\int \left [ \int f(x)\mathrm{d}x +C_{1} \right ] \mathrm{d}x +C_{2} y(n−2)=∫[∫f(x)dx+C1​]dx+C2​

依次法，连续积分 nnn 次，便得含有 nnn 个任意数的通解。

7.5.2 y′′=f(x,y′){y}'' =f(x,{y}' )y′′=f(x,y′) 型的微分方程

方程

y′′=f(x,y′){y}'' =f(x,{y}' ) y′′=f(x,y′)

右端不显含未知数 yyy 。不妨设 y′=p{y}' =py′=p ，那么上述方程就为

p′=f(x,p){p}' =f(x,p) p′=f(x,p)

这是一个关于变量 x,px,px,p 的一阶微分方程。然后通过以解为方程的解法，先求解出 ppp 的通解，然后根据 $\displaystyle y = \int p\mathrm{d}x +C_{2} $ 反解出 yyy 的通解。

7.5.3 y′′=f(y,y′){y}'' =f(y,{y}' )y′′=f(y,y′) 型的微分方程

方程

y′′=f(y,y′){y}'' =f(y,{y}' ) y′′=f(y,y′)

中不明显地含自变量 xxx 。为了求出它的解，不妨设 y′=p{y}' =py′=p ，我们可以得到：

\begin{align}{y}'' & = \frac{\mathrm{d} p}{\mathrm{d} x} \\&=\frac{\mathrm{d} p}{\mathrm{d} y}\cdot \frac{\mathrm{d} y}{\mathrm{d} x}\\&=p\frac{\mathrm{d} p}{\mathrm{d} y} \end{align}

那么，上述方程就可以转换为以下形式：

pdpdy=f(y,p)p\frac{\mathrm{d} p}{\mathrm{d} y} =f(y,p) pdydp​=f(y,p)

这是一个关于变量 y,py,py,p 的一阶微分方程。

7.6 高阶线性微分方程

7.6.2 线性微分方程的解的结构

这里讨论齐次线性方程

y′′+P(x)y′+Q(x)y=0{y}'' +P(x){y}' +Q(x)y=0 y′′+P(x)y′+Q(x)y=0

定理 1：如果函数 y1(x)y_{1} (x)y1​(x) 与 y2(x)y_{2} (x)y2​(x) 是方程的两个解，那么

y=C1y1(x)+C2y2(x)y = C_{1} y_{1} (x)+C_{2} y_{2} (x) y=C1​y1​(x)+C2​y2​(x)

也是方程的解，其中 C1,C2C_{1},C_{2}C1​,C2​ 是任意常数。

定理 2：如果函数 y1(x)y_{1} (x)y1​(x) 与 y2(x)y_{2} (x)y2​(x) 是方程的两个线性无关的特解，那么

y=C1y1(x)+C2y2(x)y = C_{1} y_{1} (x)+C_{2} y_{2} (x) y=C1​y1​(x)+C2​y2​(x)

就是方程的通解。

  • 推论：

    ​ 如果 y1(x),y2(x),…,yn(x)y_{1} (x),y_{2} (x),\dots ,y_{n} (x)y1​(x),y2​(x),…,yn​(x) 是 nnn 阶齐次线性方程

    y(n)+a1y(n−1)+⋯+an−1y′+an−1y=0{y}^{(n)} +a_{1} {y}^{(n-1)} +\dots +a_{n-1} {y}'+a_{n-1} y=0 y(n)+a1​y(n−1)+⋯+an−1​y′+an−1​y=0

    ​ 的 nnn 个线性无关的解，那么此方程的通解为

    y=C1y1(x)+C2y2(x)+⋯+Cnyn(x)y = C_{1}y_{1}(x) + C_{2}y_{2}(x)+\dots + C_{n}y_{n}(x) y=C1​y1​(x)+C2​y2​(x)+⋯+Cn​yn​(x)

    ​ 其中，C1,C2,…,CnC_{1},C_{2},\dots ,C_{n}C1​,C2​,…,Cn​ 为任意常数。

定理 3：设 $y^{*} (x) $ 是二阶非齐次线性方程

y′′+P(x)y′+Q(x)y=f(x){y}'' +P(x){y}' +Q(x)y=f(x) y′′+P(x)y′+Q(x)y=f(x)

的一个特解。 Y(x)Y(x)Y(x) 是对应的的齐次方程的通解。则二阶非线性微分方程的通解为

y=Y(x)+y∗(x)y=Y(x)+y^{*} (x) y=Y(x)+y∗(x)

定理 4：设非齐次线性方程的右端 f(x)f(x)f(x) 是两个函数之和，即

y′′+P(x)y′+Q(x)y=f1(x)+f2(x){y}'' +P(x){y}' +Q(x)y=f_{1} (x)+f_{2} (x) y′′+P(x)y′+Q(x)y=f1​(x)+f2​(x)

而 $y_{1} ^{}(x) $ 和 $y_{2} ^{}(x) $ 分别是方程

\begin{align}{y}'' +P(x){y}' +Q(x)y & = f_{1} (x)\\{y}'' +P(x){y}' +Q(x)y & = f_{2} (x)\end{align}

的特解，则 y1∗(x)+y2∗(x)y_{1} ^{*}(x)+y_{2} ^{*}(x)y1∗​(x)+y2∗​(x) 就是原方程的特解。

这一定理通常称为线性微分方程的解的**叠加定理**。

7.7 常系数齐次线性微分方程

形如： y′′+py′+qy=0{y}'' +p{y}' +qy=0y′′+py′+qy=0 的微分方程，称为二阶常系数齐次线性微分方程。

求通解的步骤：

  1. 列出特征方程，解出特征值。
    1. 特征方程： r2+pr+q=0r^{2} +pr+q=0r2+pr+q=0 ；
    2. 特征方程的解： r1=−p+p2−4q2,r2=−p−p2−4q2\displaystyle r_{1} =\frac{-p+\sqrt{p^{2}-4q} }{2} ,\quad r_{2} =\frac{-p-\sqrt{p^{2}-4q} }{2}r1​=2−p+p2−4q​​,r2​=2−p−p2−4q​​ 。
  2. 根据特征值的解的情况，列出对应的通解一般式
    1. 当 Δ=p2−4q>0\displaystyle \Delta =p^{2}-4q>0Δ=p2−4q>0 时，通解的形式为： $\displaystyle y = C_{1}e^{r_{1} x}+C_{2}e^{r_{2} x} $ 。
    2. 当 Δ=p2−4q=0\displaystyle \Delta =p^{2}-4q=0Δ=p2−4q=0 时， $\displaystyle r_{1}=r_{2} $ ，那么通解的形式为： $\displaystyle y = (C_{1}+C_{2})e^{rx} $ 。
    3. 当 Δ=p2−4q<0\displaystyle \Delta =p^{2}-4q<0Δ=p2−4q<0 时， $\displaystyle r_{1}=\alpha +\beta i ,\quad r_{2}=\alpha -\beta i $ ，那么通解的形式为： y=eα(C1cos⁡βx+C1sin⁡βx)y = e^{\alpha } (C_{1}\cos \beta x+C_{1}\sin \beta x)y=eα(C1​cosβx+C1​sinβx) 。
