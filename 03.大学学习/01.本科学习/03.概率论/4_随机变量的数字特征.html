第四章 随机变量的数字特征【概率论与数理统计】

一、数学期望

1.1 离散型的期望

定义：离散型随机变量的概率分布为：$P(X=x_{k} )=p_{k} $ ，如果级数 $\sum x_{k} p_{k} $ 绝对收敛，那么期望为：

EX=∑k=1∞xkpkEX = \displaystyle \sum_{k=1}^{\infty } x_{k} p_{k} EX=k=1∑∞​xk​pk​

1.2 连续性的期望

定义：连续型随机变量 XXX ，对应的概率密度为 f(x)f(x)f(x) ，如果 ∫−∞+∞xf(x)dx\int_{-\infty }^{+\infty } xf(x)\mathrm{d}x∫−∞+∞​xf(x)dx 绝对收敛，那么期望为：

EX=∫−∞+∞xf(x)dxEX=\int_{-\infty }^{+\infty } xf(x)\mathrm{d}x EX=∫−∞+∞​xf(x)dx

期望不一定存在。

1.3 随机变量函数的期望

离散型：

EY=∑g(xi)piEY=\sum g(x_{i} )p_{i} EY=∑g(xi​)pi​

连续性随机变量 Y=g(X)Y=g(X)Y=g(X) ，XXX 的概率密度为 f(x)f(x)f(x) ，那么

EY=∫g(x)f(x)dxEY=\int g(x)f(x)\mathrm{d}x EY=∫g(x)f(x)dx

二维随机变量函数 Z=(X,Y)Z=(X,Y)Z=(X,Y) 的期望：

  1. 离散： EZ=∑i∑jg(xi,yj)pij\displaystyle EZ =\sum_{i}\sum_{j}g(x_{i},y_{j})p_{ij}EZ=i∑​j∑​g(xi​,yj​)pij​ ；
  2. 连续性： EZ=∫−∞+∞∫−∞+∞g(xi,yj)pij\displaystyle EZ =\int_{-\infty }^{+\infty }\int_{-\infty }^{+\infty }g(x_{i},y_{j})p_{ij}EZ=∫−∞+∞​∫−∞+∞​g(xi​,yj​)pij​ 。

1.4 数学期望的性质

  1. EC=CEC=CEC=C ，即常数的期望等于常数；
  2. E(X+C)=EX+EC=EX+C\displaystyle E(X+C)=EX+EC=EX+CE(X+C)=EX+EC=EX+C ；
  3. E(CX)=C⋅EX\displaystyle E(CX)=C\cdot EXE(CX)=C⋅EX ；
  4. E(aX+b)=kEX+b\displaystyle E(aX+b) = kEX+bE(aX+b)=kEX+b ；
  5. E(X±Y)=EX±EY\displaystyle E(X\pm Y) = EX\pm EYE(X±Y)=EX±EY （任何时候都成立）；
  6. 如果 X,YX,YX,Y 独立，那么 E(XY)=EX⋅EY\displaystyle E(XY) = EX\cdot EYE(XY)=EX⋅EY ；

二、方差

2.1 方差的定义

期望是衡量数据的平均值；而期望是反映数据的集中程度。

定义（方差）： $\displaystyle DX=E(X-EX)^{2} $ ，这里注意数据的量纲

标准差： $\displaystyle \sqrt{DX} $ 。

离散型的方差：

DX=∑k(xk−EX)2pk\displaystyle DX = \sum _{k}(x_{k}-EX)^{2}p_{k} DX=k∑​(xk​−EX)2pk​

连续型的方差：

DX=∫−∞+∞(xk−EX)2f(x)dx\displaystyle DX = \int_{-\infty }^{+\infty } (x_{k}-EX)^{2}f(x)\mathrm{d}x DX=∫−∞+∞​(xk​−EX)2f(x)dx

方差的计算： $\displaystyle DX = EX^{2} -(EX)^{2} $ 。

2.2 方差的性质

  1. DC=0DC=0DC=0 ；
  2. D(X+C)=0\displaystyle D(X+C)=0D(X+C)=0 ；
  3. D(CX)=C2⋅DX\displaystyle D(CX)=C^{2} \cdot DXD(CX)=C2⋅DX ；
  4. D(kX+b)=k2DX\displaystyle D(kX+b) = k^{2} DXD(kX+b)=k2DX ；
  5. 如果 X,Y\displaystyle X,YX,Y 独立，那么 D(X±Y)=DX+DY\displaystyle D(X\pm Y)=DX+DYD(X±Y)=DX+DY ；

四、协方差

4.1 协方差的定义

定义：随机变量 X,YX,YX,Y 的协方差为

\begin{align}\displaystyle \mathrm{Cov}(X,Y) & = E\left [ \left ( X-EY \right ) \left ( Y-EX \right ) \right ] \\&=E(XY)-EX\cdot EY\end{align}

并且有：

D(X±Y)=DX+DY±Cov(X,Y)\displaystyle D(X\pm Y)=DX+DY\pm \mathrm{Cov}(X,Y) D(X±Y)=DX+DY±Cov(X,Y)

4.2 协方差的性质

  1. Cov(X,Y)=Cov(Y,X)\displaystyle \mathrm{Cov}(X,Y)=\mathrm{Cov}(Y,X)Cov(X,Y)=Cov(Y,X) ；
  2. Cov(aX,bY)=ab⋅Cov(Y,X)\displaystyle \mathrm{Cov}(aX,bY)=ab\cdot \mathrm{Cov}(Y,X)Cov(aX,bY)=ab⋅Cov(Y,X) ；
  3. Cov(X1+X2,Y)=Cov(X1,Y)+Cov(X2,Y)\displaystyle \mathrm{Cov}(X_{1}+X_{2},Y)=\mathrm{Cov}(X_{1},Y)+\mathrm{Cov}(X_{2},Y)Cov(X1​+X2​,Y)=Cov(X1​,Y)+Cov(X2​,Y) ；
  4. Cov(C,Y)=0\displaystyle \mathrm{Cov}(C,Y)=0Cov(C,Y)=0 ；
  5. 如果 X,YX,YX,Y 独立，那么 Cov(X,Y)=0\displaystyle \mathrm{Cov}(X,Y)=0Cov(X,Y)=0 。

协方差的大小，跟计量单位有关系，因此，为了反应不同单位下的随机变量的关系，就需要对随机变量进行标准化：

X⋆=X−EXDX,Y⋆=Y−EYDY\displaystyle X^{\star } =\frac{X-EX}{\sqrt{DX} } ,Y^{\star } =\frac{Y-EY}{\sqrt{DY} } X⋆=DX​X−EX​,Y⋆=DY​Y−EY​

所以，标准化后的协方差为：

Cov(X⋆,Y⋆)=Cov(X,Y)DX⋅DY=ρ\displaystyle \mathrm{Cov}(X^{\star },Y^{\star }) = \frac{\mathrm{Cov}(X,Y)}{\sqrt{DX\cdot DY} } =\rho Cov(X⋆,Y⋆)=DX⋅DY​Cov(X,Y)​=ρ

并且引入相关系数 $\rho $ 。

五、相关系数

定义：相关系数为：

ρ=Cov(X,Y)DX⋅DY=E(XY)−EX⋅EYDX⋅DY\displaystyle \rho = \frac{\mathrm{Cov}(X,Y)}{\sqrt{DX\cdot DY} }=\frac{E(XY)-EX\cdot EY}{\sqrt{DX\cdot DY} } ρ=DX⋅DY​Cov(X,Y)​=DX⋅DY​E(XY)−EX⋅EY​

性质：

  1. ∣ρ∣≤1\displaystyle \left | \rho \right | \le 1∣ρ∣≤1 ；

定理：∣ρ∣=1\displaystyle \left | \rho \right | =1∣ρ∣=1 $\displaystyle \Longleftrightarrow $ X,YX,YX,Y 成线性相关

  1. 如果 ρ=1\displaystyle \rho =1ρ=1 ，那么 X,YX,YX,Y 完全正相关；
  2. 如果 ρ=−1\displaystyle \rho =-1ρ=−1 ，那么 X,YX,YX,Y 完全负相关；
  3. ∣ρ∣\displaystyle \left | \rho \right |∣ρ∣ 接近 0 ，那么 X,YX,YX,Y 线性关系很弱；
  4. 如果 ρ=0\displaystyle \rho =0ρ=0 ，那么 X,YX,YX,Y 不存在线性关系。

独立指的是没有任何关系，包括线性关系和非线性关系

六、中心距与原点矩

原点矩： $\displaystyle EX^{k} $ 为期望 EXEXEX 的 kkk 阶原点矩；

中心距： $\displaystyle E(X-EX)^{k} $ ；

一阶中心矩为 0 ；二阶中心矩为方差。
