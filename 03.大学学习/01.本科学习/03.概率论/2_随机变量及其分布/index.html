第二章 随机变量及其分布【概率论与数理统计】

一、随机变量及其概率分布

1.1 随机变量的定义

随机变量表示随机试验各种结果的实值单值函数。随机事件不论与数量是否直接有关，都可以数量化，即都能用数量化的方式表达。

定义（随机变量）：定义在样本空间上 $\Omega $ 上的函数 X=X(ω)X=X(\omega )X=X(ω) 称为随机变量。这种函数的取值是不能预先确定的，是随机的。

从随机变量数量的角度，还能将随机变量划分为离散型随机变量与连续型随机变量，前者试验结果可取的值为有限或无限可列，后者试验结果的事件域可以与某区间 $\left [ a,b \right ] $ 或者 $\left ( -\infty,+\infty \right ) $ 构成双射。

1.2 离散型随机变量及其概率分布

假设抛骰子，那么随机变量的所有可能的取值是： 1,2,3,4,5,61,2,3,4,5,61,2,3,4,5,6 。

定义：随机变量 XXX 的所有可能的取值 xk(k=1,2,⋯ ,)x_{k}(k=1,2,\cdots ,)xk​(k=1,2,⋯,) ，那么对应的概率就是

P(X=xk)=pkP(X=x_{k})=p_{k} P(X=xk​)=pk​

上述概率称为概率函数或者概率分布。

性质：

  1. pk≥0p_{k} \ge 0pk​≥0 ；
  2. ∑pk=1\sum p_{k}=1∑pk​=1 。

概率分布的表示方法：

  • 表格；
  • 数据函数；
  • 图像。

  4. $\displaystyle P(a\le x\le b)=\int_{-\infty }^{b} f(x)\mathrm{d}x -\int_{-\infty }^{a} f(x)\mathrm{d}x=\int_{a }^{b} f(x)\mathrm{d}x $ 。

二、分布函数

定义（分布函数）：定义函数 F(x)=P(X≤x)F(x)=P(X\le x)F(x)=P(X≤x) 为分布函数，其含义为 XXX 取值不超过 xxx 的概率。

  1. $x\in \left ( -\infty ,+\infty \right ) $ ；
  2. $F(x)\in \left [ 0,1 \right ] $ ；
  3. $\displaystyle F(x) = P(X\le x)=\int_{-\infty }^{x} f(t)\mathrm{d}t $ 。

性质 3：分布函数 F(x)F(x)F(x) 是右连续的，且连续性随机变量的分布函数是连续的。

  • 证明（右连续，即 lim⁡x→a+F(x)=F(a)\displaystyle \lim_{x \to a^{+} } F(x)=F(a)x→a+lim​F(x)=F(a) ）：

  4. P(X=a)=F(a)−F(a−)\displaystyle P(X=a)=F(a)-F(a^{-} )P(X=a)=F(a)−F(a−) ；

三、常见随机变量的分布

3.1 0-1分布

形如

X 1 0
P p 1-p

的分布形式，称为0-1分布。

定义（0-1分布）： P(X=k)=pk(1−p)1−k(k=1,2)P(X=k)=p^{k} (1-p)^{1-k} \quad (k=1,2)P(X=k)=pk(1−p)1−k(k=1,2) 。

  • 只有两个试验结果；
  • 试验只进行一次

数学特征：

  • 期望： EX=p\displaystyle EX=pEX=p ；
  • 方差： DX=2p(1−p)\displaystyle DX=2p(1-p)DX=2p(1−p) 。

3.2 几何分布

定义（几何分布）：在一次试验中，事件 AAA 发生的概率是 ppp ，那么在 nnn 次独立重复试验中，第 kkk 次试验中，事件 AAA 第一次发生的概率为

P(X=k)=(1−p)k−1pP(X=k)=(1-p)^{k-1} p P(X=k)=(1−p)k−1p

上述分布称为几何分布，记作 X∼G(p)X\sim G(p)X∼G(p) 。

  • 期望： $\displaystyle E(X)=\frac{1}{p} $ ；
  • 方差： $\displaystyle DX=\frac{1-p}{p^{2} } $ 。

3.3 二项分布

定义（二项分布）：在一次试验中，事件 AAA 发生的概率是 P(A)=pP(A)=pP(A)=p ，那么，在 nnn 次独立重复实验中，事件 AAA 发生了 kkk 次，其发生的次数与其概率的关系为：

P(X=k)=Cnkpk(1−p)n−k(k=0,1,2,⋯ ,n)\displaystyle P(X=k)=C_{n} ^{k} p^{k} (1-p)^{n-k} \quad (k =0, 1,2,\cdots ,n) P(X=k)=Cnk​pk(1−p)n−k(k=0,1,2,⋯,n)

上述分布称为二项分布，记作 X∼B(n,p)X\sim B(n,p)X∼B(n,p) 。

  • 期望： EX=np\displaystyle EX=npEX=np ；
  • 方差： DX=np(1−p)\displaystyle DX=np(1-p)DX=np(1−p) 。

3.4 泊松分布

定义（泊松分布）：形如

P(X=k)=λkk!e−λ\displaystyle P(X=k)=\frac{\lambda ^{k} }{k!} e^{-\lambda } P(X=k)=k!λk​e−λ

的分布律称为泊松分布，记作 X∼P(λ)X\sim P(\lambda )X∼P(λ) 。

  • 期望： EX=λEX=\lambdaEX=λ ；
  • 方差： DX=λDX=\lambdaDX=λ 。

为什么需要泊松分布？或者它的意义是什么？

泊松分布源于二项分布，二项分布的性质是 μ=npμ = npμ=np 。可以这么理解：现实中虽然只进行了一回 n 次试验，但是假想平行时间线上进行了无数回 n 次试验（直接多做几回 n 次试验），将这无数时间线上的成功次数 k 做算术平均，得到的均值就是μ。

泊松分布的一个作用就是近似：

当二项分布 X∼B(n,p)X\sim B(n,p)X∼B(n,p) 中的 nnn 很大，ppp 很小，并且 npnpnp 的大小适中，就能用泊松分布来近似计算，即 X∼P(np)X\sim P(np)X∼P(np) 。

定义（超几何分布）：假设 NNN 个元素分成两类，第一类有 M(M≤N)M(M\le N)M(M≤N) 个，第二类有 N−MN-MN−M 个，那么取出 nnn 个元素，随机变量 XXX 为取出的第一类元素的个数，那么其概率为：

P(X=k)=CMkCN−Mn−kCNn\displaystyle P(X=k)=\frac{C_{M}^{k} C_{N-M}^{n-k} }{C_{N}^{n} } P(X=k)=CNn​CMk​CN−Mn−k​​

当 $N\to \infty $ 时，超几何分布近似为二项分布， MN→p\displaystyle \frac{M }{N} \to pNM​→p 。
