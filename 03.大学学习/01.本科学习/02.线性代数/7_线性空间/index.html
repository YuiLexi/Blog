第七章 线性空间【线性代数】

一、线性空间

1.1 数域和元素

定义（数域）：设 FFF 是一个至少包含数 0 和 1 的集合，如果 FFF 中任意两个数的和、差、积、商（除数不为 0 ）都在 FFF 中，则称 FFF 是一个**数域**。

有理数集 QQQ 、实数集 RRR、复数集 CCC 都是数域，他们分别称为有理数域、实数域、复数域。

若集合 VVV 中定义了某种运算， VVV 中的任意元素进行运算所得到的结果均在 VVV 中，则称这种运算是封闭的。

1.2 线性空间的定义

定义：设 VVV 是一个非空集合。其元素用 $\displaystyle \alpha ,\beta, \gamma ,\delta ,\dots $ 向量表示，FFF 是一个数域，其元素用 $\displaystyle k,l,m,n,\dots $ 表示。并且满足下面条件：

  1. 在集合 VVV 的元素之间定义了一种运算，叫做加法。即对于 VVV 中任意两个元素 α\alphaα 和 $\beta $ ，在 VVV 中都有唯一的元素 $\gamma $ 与之对应，称为 α\alphaα 和 $\beta $ 的和，记作 $\gamma = \alpha +\beta $ 。
  2. 在数域 FFF 的数与集合 VVV 的元素之间定义了一种运算，叫做数量乘法，也就是说，对于 FFF 任意一个数 kkk 与 VVV 中任一元素 α\alphaα ，在 VVV 中都有唯一的元素 $\delta $ 与之对应，$\delta = k\alpha $ 称为 kkk 与 α\alphaα 的数量积。
  3. VVV 中定义的加法与数量乘法满足下列运算律：
    1. $\alpha +\beta = \beta +\alpha $ ；
    2. (α+β)+γ=α+(β+γ)(\alpha +\beta)+ \gamma = \alpha +(\beta +\gamma )(α+β)+γ=α+(β+γ) ；
    3. 在 VVV 中有一个元素 0 ，对 VVV 中任一元素 α\alphaα ，都有 $\alpha +0 = \alpha $ （具有这性质的元素称为零元素）；
    4. 对于 VVV 中任一元素 α\alphaα ，都存在 VVV 中元素 $\beta $ ，使 α+β=0\alpha +\beta =0α+β=0 ，称 $\beta $ 为 α\alphaα 的负元素；
    5. $k(\alpha +\beta ) = k\alpha +k\beta $ ；
    6. $(k+l)\alpha = k\alpha +l\alpha $ ；
    7. $k(l\alpha ) = (kl)\alpha $ ；
    8. $1\alpha = \alpha $ ；

则称 VVV 为数域 FFF 上的线性空间（或向量空间），简称 VVV 的线性空间或向量空间。

VVV 中所定义的加法以及数量乘法运算统称为 VVV 的线性运算。 VVV 中的元素也称为向量， FFF 中的元素也称为数量或标量。当 FFF 为实数域 RRR 时，称 VVV 为实线性空间；当 FFF 为复数域 CCC 时，称 VVV 为复线性空间。

1.3 线性空间的性质

性质：零向量是唯一的。

性质：向量空间 VVV 中的每个向量的复向量是唯一的。

性质：对向量空间 VVV 中任一向量 $\alpha $ 与任一数 kkk ，有 0α=00\alpha =00α=0 ；k0=0k0=0k0=0 ；$(-1)\alpha =\alpha $ 。

1.4 线性子空间

定义：若 WWW 为数域 FFF 上的线性空间 VVV 的非空子集，且 WWW 满足如下条件：

  1. 对任意的 k∈Fk\in Fk∈F ，a∈Wa\in Wa∈W ，则 kα∈Wk\alpha \in Wkα∈W ；
  2. 对任意的 α,β∈W\alpha ,\beta \in Wα,β∈W ，则 α+β∈W\alpha+\beta \in Wα+β∈W ；

则称 WWW 为 VVV 的线性子空间，简称子空间。换句话说，当且仅当 WWW 对 VVV 中的加法与数乘运算是封闭的，WWW 是向量空间 VVV 的子空间。

若 WWW 是线性空间 VVV 的子空间，由定义不难得出，它也是线性空间。

二、线性空间的基与向量坐标

2.1 基、维数、坐标

定义：设 $\displaystyle a_{1} ,a_{2} ,\dots, a_{m} $ 是数域 FFF 上的线性空间 VVV 中的 mmm 个向量， $k_{1} ,k_{2} ,\cdots ,k_{m} $ 是 FFF 中的数，则称 $k_{1}a_{1}+k_{2}a_{2}+\cdots +k_{m}a_{m} = \beta $ 为向量组 $\displaystyle a_{1} ,a_{2} ,\dots, a_{m} $ 的一个线性组合，或者 $\beta $ 可由向量组 $\displaystyle a_{1} ,a_{2} ,\dots, a_{m} $ 线性表示。

定义：设 $\displaystyle a_{1} ,a_{2} ,\dots, a_{m} $ 是数域 FFF 上的线性空间 VVV 中向量，若存在 FFF 中不全为零的数 k1,k2,…,kmk_{1},k_{2},\dots ,k_{m}k1​,k2​,…,km​ ，使 k1a1+k2a2+⋯+kmam=0k_{1}a_{1}+k_{2}a_{2}+\cdots +k_{m}a_{m} = 0k1​a1​+k2​a2​+⋯+km​am​=0 ，则称向量组 $\displaystyle a_{1} ,a_{2} ,\dots, a_{m} $ 线性相关；否则称向量组线性无关。

定义：设 VVV 是数域 FFF 上的线性空间， $\displaystyle a_{1} ,a_{2} ,\dots, a_{n} $ 是 VVV 中 nnn 个线性无关的向量，若 VVV 中任一向量 aaa 均可由 $\displaystyle a_{1} ,a_{2} ,\dots, a_{n} $ 线性表示，则称线性空间 VVV 是 nnn 维线性空间， nnn 称为 VVV 的维数，记作： dimV=n\text{dim} V = ndimV=n ，而称 $\displaystyle a_{1} ,a_{2} ,\dots, a_{n} $ 为 VVV 的一组基。如果 VVV 中有任意多个线性无关向量，则称 $$V$$ 是无限维线性空间。

定义：设 $\displaystyle a_{1} ,a_{2} ,\dots, a_{n} $ 是 nnn 维线性空间 VVV 的一组基，β∈V\beta \in Vβ∈V 且有

β=x1a1+x2a2+⋯+xnan\beta = x_{1}a_{1}+x_{2}a_{2} +\dots +x_{n}a_{n} β=x1​a1​+x2​a2​+⋯+xn​an​

即：

  • 左边：线性空间 VVV 中的任一向量（元素）；
  • 右边：线性空间 VVV 的一组基构成的基矩阵乘以向量在该基下的坐标。

三、线性变换

3.1 映射

定义 5.8：设 MMM 与 NNN 是两个非空集合，如果有一个法则 fff ，使得对 MMM 中的每个元素 aaa 都有 NNN 中唯一确定的 bbb 与之对应，那么就称 fff 是 MMM 到 NNN 的一个映射，记作

f:M→Nf:\mathbf{M} \to \mathbf{N} f:M→N

并称 bbb 为 aaa 在 fff 下的像，而 aaa 称为 bbb 在映射 fff 下的一个原像，记作

f:a→b或f(a)=bf:a\to b\quad \textbf{或} f(a) = b f:a→b或f(a)=b

集合 MMM 称为映射 fff 的定义域。如果对于任意的 a,b∈M\displaystyle a,b\in Ma,b∈M ，都有 f(a)≠f(b)(a≠b)f(a)\ne f(b)\quad (a\ne b)f(a)=f(b)(a=b) ，则称 fff 是单射；如果对于任意的 b∈N,a∈Mb\in N,a\in Mb∈N,a∈M ，都有 f(a)=bf(a)=bf(a)=b ，则称 fff 是满射；如果 fff 既是单射又是满射，则称 fff 是双射（或称为一一映射）。

3.2 线性变换的定义

定义：设数域 FFF 上的线性空间 $V_{1} $ 和 $V_{2} $ 分别是 nnn 和 mmm 维的。 TTT 是 $V_{1} $ 到 $V_{2} $ 的一个映射，且满足：

  1. 对任意的 $\alpha ,\beta \in V_{1} $ ，有 T(α+β)=T(α)+T(β)T(\alpha +\beta )=T(\alpha )+T(\beta )T(α+β)=T(α)+T(β) ；

只有高维空间投影到低维空间，而不能从低维空间或投影到高维空间，也就是说，只能有高维观测低位而不能有低维观测高维。

5.1 特征值与特征向量

定义：设 TTT 是数域 FFF 上线性空间 VVV 的线性变换，若存在数 $\lambda \in \mathbf{F} $ 和非零向量 α∈V\alpha \in Vα∈V ，使得

T(α)=λαT(\alpha ) = \lambda \alpha T(α)=λα

则称 $ \lambda$ 为 TTT 的一个特征值，而 $\alpha $ 称为 TTT 属于特征值 $\lambda $ 的特征向量。

**定理 **：设 TTT 是 nnn 维线性空间 VVV 上的线性变换，TTT 在 VVV 的基 α1,α2,…,αn\alpha _{1},\alpha _{2},\dots ,\alpha _{n}α1​,α2​,…,αn​ 下的矩阵为 AAA ，则：

  1. $\lambda $ 是 TTT 的特征值得充要条件是$\lambda $ 为 AAA 的特征值；
  2. α∈V\alpha \in Vα∈V 是 TTT 的属于特征值 $\lambda $ 的特征向量的充要条件是 $\alpha $ 在基 α1,α2,…,αn\alpha _{1},\alpha _{2},\dots ,\alpha _{n}α1​,α2​,…,αn​ 下的坐标是矩阵 AAA 的属于特征值 $\lambda $ 的特征向量。

5.2 值域与核

定义：设 TTT 是线性空间 VVV 中的线性变换，VVV 中所有元素在 TTT 下的像所组成的集合。

{β∣β=T(α),α∈V}\left \{ \beta |\beta = T(\alpha ) ,\alpha \in V\right \} {β∣β=T(α),α∈V}

称为 TTT 的值域，记作 ImT或者T(V)\text{Im} T\quad \text{或者} T(V)ImT或者T(V) 。零向量的所有原像的集合：

{α∣T(α)=0,α∈V}\left \{ \alpha | T(\alpha )=0 ,\alpha \in V\right \} {α∣T(α)=0,α∈V}

称为 TTT 的核，记作 KerT或者T−1(V)\text{Ker} T\quad \text{或者} T^{-1} (V)KerT或者T−1(V) 。

关于线性变换的值域与核有以下性质：

  1. 线性变换 TTT 的值域 ImT\text{Im} TImT 是 VVV 的子空间；
  2. 线性变换 TTT 的核 KerT\text{Ker} TKerT 是 VVV 的子空间。

定理：设 VVV 是 nnn 维线性空间， VVV 上的线性变换 TTT 在 VVV 的基 α1,α2,…,αn\alpha _{1},\alpha _{2},\dots ,\alpha _{n}α1​,α2​,…,αn​ 下的矩阵 AAA ，则

  1. ImT\text{Im} TImT 的维数等于向量组 T(α1),T(α2),…,T(αn)T(\alpha_{1} ),T(\alpha_{2} ),\dots ,T(\alpha_{n} )T(α1​),T(α2​),…,T(αn​) 的秩，也等于矩阵 AAA 的秩；
  2. ImT\text{Im} TImT 的基是向量组 T(α1),T(α2),…,T(αn)T(\alpha_{1} ),T(\alpha_{2} ),\dots ,T(\alpha_{n} )T(α1​),T(α2​),…,T(αn​) 的一个极大无关组。

定理：设 VVV 是 nnn 维线性空间， VVV 上的线性变换 TTT 在 VVV 的基 α1,α2,…,αn\alpha _{1},\alpha _{2},\dots ,\alpha _{n}α1​,α2​,…,αn​ 下的矩阵为 AAA ，则

  • TTT 的核 KerT\text{Ker} TKerT 的维数等于 n−R(A)n-R(A)n−R(A) ；
  • VVV 中齐次线性方程组 Ax=0Ax = 0Ax=0 的基础解系构成 TTT 的核 KerT\text{Ker} TKerT的基。
